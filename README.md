# UNR CS 474 - Image Processing

## Building with CMake
```bash
mkdir build
cd build
cmake ..
make
```

## Executables
* ./Threshold 
  * Example threshold program given in class
* ./PA01_Equalization
  *  For image equalization
* ./PA01_Sampling
  *  For image sampling (changing spacial resolution)
* ./PA01_Quantization
  *  For image quantization (reducing gray levels)
